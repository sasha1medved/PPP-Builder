How to complete the PPP
  1. Fill in Project info first.
  2. Add one clear photo for Finished packaging and one for Parts & materials.
  3. Add each packaging material.
  4. Create packing steps by uploading one readable photo per step or all at once.
  5. Follow steps below to enable AI image description tool.
  6. Use Generate All Descriptions to fill all image descriptions at once, or use Generate description for one section only.
  7. If you need to correct an existing PPP, click Load file and open a previously exported PPTX, PDF, or JSON draft.
  8. Make your updates and re-export the file or
  9. Click Submit for Review, then paste (or reuse) SharePoint folder link and upload the exported file in that folder for review.
